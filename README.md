# ImageSucker  图片吮吸者
Picture Converter with Python 图片转换工具  
**中文的图形界面， 中文的README**
## QuickStart
### 左侧栏 FileBox
是当前文件夹的文件列表，需要保持高亮选定

### 右侧栏 尺寸
第一个Entry 为width， 第二个Entry 为heigh  
不能缺省

### 右侧栏 格式
不可以加`.`， 可以缺省，缺省值为`jpeg`  
常见的图形格式都可以，如 ： `png`, `jpg`, `bmp`等  
**会把`jpg`后缀改为`jpeg`**  

### 右侧栏 六个常用尺寸
添加了六个常用的尺寸，选定文件后直接按Button即可  

## 保存路径
一般保存在当前目录
